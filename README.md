**Скрипт для инвентаризации хостов ESXI**

Cбор данных происходит по ssh, предварительно надо включить эту службу на хосте ESXI.

Переменная "HOST_CREDS" определяет адрес  хоста  и пароль пользователя, пользователь определяется в переменной "USER"

Запускаем скрипт при помощи bash : ```bash inventory_esxi_vm_to_csv.sh```

После выполнения скрипта, в этой же папке появляется отечет  ESXI_VM_report.csv, со всей собранной информацией
